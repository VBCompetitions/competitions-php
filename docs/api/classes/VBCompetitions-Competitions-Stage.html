public getTeamIDs([int $flags = VBC_TEAMS_FIXED_ID ]) : array<string|int, string>
Parameters
$flags : int = VBC_TEAMS_FIXED_ID

Controls what gets returned (MAYBE overrides KNOWN overrides FIXED_ID)

  • VBC_TEAMS_FIXED_ID (default) returns teams with a defined team ID (no references)
  • VBC_TEAMS_KNOWN returns teams that are known (references that are resolved)
  • VBC_TEAMS_MAYBE returns teams that might be in this container (references that may resolve to a team)
  • VBC_TEAMS_ALL returns all team IDs, including unresolved references

teamMayHaveMatches()

Stage.php : 642

Check if a team may have matches scheduled in any group within this stage. Note that this has undefined behaviour if a team definitely has matches, i.e. if you want to know if a team definitely has matches in this stage then call teamHasMatches(), but if you want to know if there are any references that might point to this team (e.g. a reference to a team in a league position in an incomplete league) then call this function. This is essentially so we know whether we still need to consider displaying a stage to the competitors as they <i>might</i> still be able to reach that stage.
